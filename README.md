# 📊 E-Commerce Sales Dashboard Analysis

## 📝 Project Overview
This project presents an interactive **E-Commerce Sales Dashboard** built using **Power BI** to analyze sales performance, customer behavior, profit trends, and regional insights.

The dashboard converts raw ecommerce sales data into meaningful business insights for better decision-making.

---

## 🚀 Features
- Sales Analysis Dashboard
- Profit & Loss Analysis
- Customer Insights
- Category Performance Tracking
- Regional Sales Analysis
- Payment Mode Analysis
- Interactive Filters & Slicers

---
